## OpenAI — TTS

### Актуальные цены

| Наш ID     | Модель           | Цена                                            | Наша цена         |
| ---------- | ---------------- | ----------------------------------------------- | ----------------- |
| tts-openai | tts-1 (Standard) | **$15.00/1M chars** ($0.015/1K)                 | $0.0045/req ✅    |
| —          | tts-1-hd (HD)    | **$30.00/1M chars** ($0.030/1K)                 | не поддерживается |
| —          | gpt-4o-mini-tts  | **$0.60/1M input tokens + $12/1M audio tokens** | не поддерживается |

**Расчёт текущей цены:** 300 символов × $0.015/1K = $0.0045 ✅ — точно для tts-1.

**Важно:** у нас нет настройки выбора модели. Мы всегда используем `tts-1`. Добавив выбор модели, можно предложить HD ($2× дороже, выше качество) или gpt-4o-mini-tts (управляемый через инструкции, другая тарификация).

### Голоса

Текущие у нас (9 голосов): alloy, ash, coral, echo, fable, nova, onyx, sage, shimmer.

**Недостающие голоса (добавить 2):** `ballad`, `verse` — новые голоса, доступны для всех TTS моделей.

Итого должно быть 11: alloy, ash, **ballad**, coral, echo, fable, nova, onyx, sage, shimmer, **verse**.

### Дополнительные настройки для добавления

**Выбор модели TTS:**

```
key: "model"
label: "Модель TTS"
description: "tts-1 — стандартное качество ($0.015/1K chars), tts-1-hd — высокое качество ($0.030/1K chars), gpt-4o-mini-tts — управляемый стиль речи через инструкции."
type: "select"
options: [tts-1, tts-1-hd, gpt-4o-mini-tts]
default: "tts-1"
```

**Инструкции для gpt-4o-mini-tts:**

```

label: "Инструкции к голосу"
description: "Только для gpt-4o-mini-tts: укажите тон, эмоцию и стиль речи. Например: 'Говори медленно и торжественно' или 'Эмоциональный диктор новостей'."
type: "text"
default: ""
```

## ElevenLabs — Клонирование голоса

### Актуальные цены

ElevenLabs использует кредитную систему с тиерами. Для API-доступа:

| Тир                  | Цена за 1K символов | ~300 символов |
| -------------------- | ------------------- | ------------- |
| Creator ($11/мес)    | $0.30/1K            | $0.090/req    |
| Pro ($99/мес)        | $0.24/1K            | $0.072/req    |
| Scale ($330/мес)     | $0.18/1K            | $0.054/req    |
| Business ($1320/мес) | $0.12/1K            | $0.036/req    |

Наша цена $0.072/req соответствует Pro-тиру ✅.

**Критично:** модель `eleven_turbo_v2_5` стоит **в 2× дешевле** multilingual_v2 (0.5 кредита/символ вместо 1). Это снизило бы затраты до $0.036 при Pro-тире — стоит добавить выбор модели.

| `eleven_turbo_v2_5`      | **0.5 кредита/символ** | Низкая задержка, 32 языка, ~вдвое дешевле |

## Suno — Генерация музыки

### Актуальные цены

Suno **не имеет официального API**. Мы используем прокси apipass.net.

| Источник               | Цена за трек                                    |
| ---------------------- | ----------------------------------------------- |
| Suno Premier ($30/мес) | ~$0.036/трек (10K кредитов, v5 = 12 кредитов)   |
| sunoapi.org            | ~$0.005/кредит; стоимость трека не раскрывается |
| apipass.net            | точная стоимость не публикуется                 |

Наша цена $0.035/req — **разумная оценка** ✅.

### Актуальные версии моделей Suno

| Версия          | Кредиты/трек | Длина    | Примечание             |
| --------------- | ------------ | -------- | ---------------------- |
| chirp-v4        | 8            | до 4 мин | у нас сейчас           |
| chirp-v4.5      | 10           | до 8 мин | лучше качество         |
| chirp-v4.5 Plus | 10           | до 8 мин | расширенный стиль      |
| **v5**          | **12**       | до 8 мин | **текущая актуальная** |

**Критично:** в адаптере у нас задано `mv: "chirp-v4"` — устаревшая версия. Стоит обновить на `chirp-v4.5` или `v5`.

### Дополнительные настройки для добавления

**Версия модели:**

**Пользовательский текст песни:**

default: ""
