## 5. Гипотеза о прибыльности приложения

### Если пользователи используют преимущественно чат (>80% трафика)
Приложение **прибыльно**. Валовая маржа LLM-сегмента перекрывает операционные расходы.
Типичный пользователь тратит 10 токенов/месяц в чате → прибыль ~$0.37.

### Если пользователи активно используют медиа-генерацию
Приложение **быстро уходит в минус**:
- Один видеоклип Kling = убыток $0.31–$0.66, перекрываемый ~7–15 GPT-запросами
- Активный видеопользователь (10 клипов/день) = −$3–$7/день → −$90–$210/месяц
- Один такой пользователь уничтожает прибыль с ~200 чат-пользователей

### Общий вывод
Текущая модель **нежизнеспособна при активном использовании медиа**. Прибыльность держится на LLM-субсидии, но один-два "тяжёлых" медиа-пользователя могут обнулить весь доход.
